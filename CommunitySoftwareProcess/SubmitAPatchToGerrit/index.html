Submit a Patch to Gerrit¶

Description¶

Patches to the ITK repository pass through our Gerrit Code Review instance before being merge. This session describes how to create and submit a patch to Gerrit.

More information¶

The ITK patch submission process is covered concisely on the Wiki. In this session, we describe how to create and submit a patch. Other parts of the patch lifecycle, such as setting up your local repository for development, are covered in other sessions.
