Style Check ITK with KWStyle¶

Description¶

KWStyle is an executable that takes a set of style rules and exceptions and ensures source code adheres to those rules. We describe how to build KWStyle and configure the ITK build, test, and commit process to automatically verify code changes and additions are ITK style compliant.
